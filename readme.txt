This is NyARToolkitCS for Unity3D varion 4.1.0
NyARToolkitCS is C# version AR(Augmented Reality)library.



* Install

 1.Create blank project.
 2.Import package.
 3.Click "simplelite" seane.
 4.Play proglam.
 5.Print "pattHiro.pdf" and take it by webcam.

* Sample

NyARToolkit for Unity has some sample scenes.

at sample

 1.ImagePickup - A marker pattern is scraped from a image plane. 
 2.JpegFile    - A Texture is used for an input instead of a camera. 
 3.MarkerPlane - A cube walks on marker by mouse cursor.
 4.SimpleLite  - Single Marker demo.
 5.SimpleLiteM - Multiple marker demo.

at sample.webplayer
 
 1.SimpleLiteWeb - Single matker sample that supports webcamera authorization.
